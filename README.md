# ev-sales-forecasting-lstm-arima-sarima-ets
Time Series Forecasting of Global Electric Vehicle Sales using ARIMA, SARIMA, Exponential Smoothing (ETS), and LSTM Deep Learning models.
This project forecasts Global Electric Vehicle (EV) Sales using both traditional statistical forecasting methods and deep learning techniques.

Models Used:
• ARIMA
• SARIMA
• Exponential Smoothing (ETS)
• LSTM Neural Network

The objective is to compare forecasting approaches on historical EV sales data and analyze their ability to capture the rapid growth of electric vehicle adoption worldwide.
